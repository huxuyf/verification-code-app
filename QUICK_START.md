# 快速开始指南

## 首次使用步骤

### 1. 安装应用
将 APK 文件安装到您的 Android 设备上。

### 2. 启动应用
打开"验证码助手"应用。

### 3. 授予权限

#### 第一步：授予短信权限
- 点击"授予权限"按钮
- 在权限请求对话框中，点击"允许"

#### 第二步：授予悬浮窗权限
- 授予短信权限后，应用会自动跳转到系统设置
- 找到"验证码助手"应用
- 开启"在其他应用上层显示"开关

### 4. 开始使用
- 返回应用，点击"开始监听"按钮
- 应用会在后台运行
- 当收到包含验证码的短信时，会自动弹出悬浮窗
- 点击"复制验证码"即可复制

## 测试验证

### 发送测试短信
可以使用以下方式测试：
1. 让朋友发送包含验证码的短信给您
2. 使用手机自带的短信应用发送测试短信（部分设备可能不支持）

```

```

## 常用操作

### 查看权限状态
应用主界面会实时显示权限状态：
- ✓ 已授予：权限已正常授予
- ✗ 未授予：需要授予该权限

### 启动/停止监听
- **开始监听**：点击绿色按钮启动监听服务
- **停止监听**：点击红色按钮停止监听服务

### 操作悬浮窗
- **拖动**：长按悬浮窗可拖动位置
- **复制**：点击"复制验证码"按钮复制验证码
- **关闭**：点击右上角"✕"按钮关闭悬浮窗

## 故障排查

### 问题：收不到验证码弹窗

**解决方案：**
1. 检查权限是否已全部授予
2. 确认监听服务已启动
3. 检查短信内容是否包含"验证码"或"密码"关键词
4. 检查系统省电模式是否限制了后台服务

### 问题：悬浮窗无法显示

**解决方案：**
1. 进入系统设置 > 应用管理 > 验证码助手
2. 检查"在其他应用上层显示"权限是否开启
3. 重启应用

### 问题：权限无法授予

**解决方案：**
1. 进入系统设置 > 应用管理 > 验证码助手 > 权限
2. 手动开启短信和悬浮窗权限
3. 重启应用

## 省电设置建议

某些 Android 设备的省电模式可能会限制后台服务，建议：

### 小米/红米设备
1. 进入系统设置 > 省电与电池 > 应用智能省电
2. 找到"验证码助手"应用
3. 选择"无限制"或"省电模式"

### 华为/荣耀设备
1. 进入系统设置 > 应用启动管理
2. 找到"验证码助手"应用
3. 关闭"自动管理"
4. 开启"允许后台活动"

### OPPO/vivo设备
1. 进入系统设置 > 电池耗电管理
2. 找到"验证码助手"应用
3. 选择"允许完全后台行为"
