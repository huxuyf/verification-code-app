================================================================================
                              功能特性
================================================================================

核心功能：
  ✓ 短信监听 - 实时监听手机接收的短信
  ✓ 关键词匹配 - 识别包含"验证码"或"密码"的短信
  ✓ 验证码提取 - 自动提取4-8位数字验证码
  ✓ 悬浮窗显示 - 美观的悬浮窗界面，支持拖动
  ✓ 一键复制 - 点击按钮复制验证码到剪贴板
  ✓ 关闭功能 - 提供清晰的关闭按钮

权限管理：
  ✓ 动态权限申请 - 支持 Android 6.0+ 运行时权限
  ✓ 权限状态显示 - 实时显示权限授予状态
  ✓ 悬浮窗权限引导 - 引导用户授予悬浮窗权限
  ✓ 权限检查验证 - 确保所有权限已授予

用户界面：
  ✓ 主界面 - 清晰的权限状态显示和操作按钮
  ✓ 悬浮窗 - 圆角卡片设计，醒目的验证码显示
  ✓ 响应式设计 - 支持不同屏幕尺寸
  ✓ 视觉反馈 - 按钮点击效果和状态变化

服务管理：
  ✓ 前台服务 - 确保监听服务不被系统杀死
  ✓ 服务控制 - 用户可启动/停止服务
  ✓ 轻量化设计 - 避免过度消耗系统资源

隐私保护：
  ✓ 本地处理 - 所有数据在本地处理
  ✓ 无网络请求 - 不请求网络权限
  ✓ 无数据收集 - 不收集或上传任何用户数据

================================================================================
                              使用流程
================================================================================

1. 安装应用
   └─> 将 APK 安装到 Android 设备

2. 授予权限
   ├─> 授予短信权限
   │   └─> 点击"授予权限" > 允许
   └─> 授予悬浮窗权限
       └─> 系统设置 > 允许在其他应用上层显示

3. 开始使用
   ├─> 点击"开始监听"
   ├─> 应用在后台运行
   ├─> 收到验证码短信时自动弹窗
   └─> 点击"复制验证码"即可复制

4. 停止监听
   └─> 点击"停止监听"按钮

================================================================================
                              编译说明
================================================================================

环境要求：
  - Android Studio Arctic Fox 或更高版本
  - JDK 8 或更高版本
  - Android SDK API 34

编译步骤：
  1. 使用 Android Studio 打开项目
  2. 等待 Gradle 同步完成
  3. Build -> Build Bundle(s) / APK(s) -> Build APK(s)
  4. APK 位于：app/build/outputs/apk/debug/app-debug.apk

Release 打包：
  1. 配置签名信息
  2. Build -> Generate Signed Bundle / APK
  3. 选择 APK 和 release 构建类型
  4. 生成签名 APK
